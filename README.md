# Pruebas previas

Después de descargar y descomprimir este repositorio ejecuta el siguiente comando para instalar dependencias: 

```
flutter pub get
```


Hasta que llegue el momento de crear la rama `proyecto`, y con el fin de que puedas ir investigando lo que más adelante añadirás, te proporciono este **proyecto de Flutter descargable** para que puedas adelantar e investigar todo lo que quieras sin comprometer tu repositorio de clase.


Ya está configurado para poder poner imágenes locales:
- Las imágenes se encuentran dentro de la carpeta "assets/images/". 
- Se ha añadido  una linea de código en el archivo. pubspec.yaml que apunta a esa ruta.
- Revisa los ejemplod de como han sido añadidas las imágenes en las pantallas de bandas y mapas
